Handbuch für Administratoren

Alle Funktionen inkl. Systemverwaltung und Audit-Log

1. Alle HR-Funktionen

Als Admin stehen Ihnen alle Funktionen des Personalmanagements zur Verfügung (HR-Dashboard, Mitarbeiterverwaltung, Zeitkorrekturen bis 3 Monate, Genehmigungen, Berichte). Zusätzlich:

2. Zeitkorrekturen ohne Zeitlimit

  • Admins können Zeiteinträge unbegrenzt weit zurück korrigieren.
  • Jede Korrektur erfordert eine Begründung und wird im Audit-Log gespeichert.
  • Original-Zeiten bleiben in der Datenbank erhalten (§16 ArbZG Aufbewahrungspflicht 2 Jahre).

3. Audit-Log (Django Admin)

  • Unter Django Admin → Core → Audit Logs sehen Sie alle protokollierten Aktionen.
  • Protokollierte Aktionen: Ein-/Ausstempeln, Zeitkorrekturen, Genehmigungen, Ablehnungen, Mitarbeiterverwaltung, Berichtsdownloads.
  • Jeder Eintrag enthält: Akteur, Zeitstempel, IP-Adresse, betroffene Entität, alte und neue Werte.
  • Einträge sind nur lesbar – keine Änderung oder Löschung möglich.

4. Abwesenheitstypen verwalten

  • Unter Admin → Abwesenheitstypen können Sie neue Typen anlegen oder bestehende anpassen.
  • Vorhandene Typen: Urlaub, Krank, Sonderurlaub, Unbezahlter Urlaub, Überstundenausgleich.

5. Feiertage pflegen

  • Unter Admin → Core → Feiertage können Feiertage pro Bundesland gepflegt werden.
  • Befehl zum Importieren: python manage.py seed_holidays

6. Rollen vergeben

  • Rollen werden im Mitarbeiterprofil (HR-Bereich) oder unter Admin → Mitarbeiterprofile vergeben.
  • Mitarbeiter: Basiszugriff (Self-Service)
  • Personalmanagement: HR-Dashboard, Mitarbeiterverwaltung, Genehmigungen
  • Admin: Alles + Audit-Log + Django Admin

7. Endgültige Datenlöschung (DSGVO Art. 17)

  • Datenlöschung auf Antrag (Recht auf Vergessenwerden) nur durch Admin möglich.
  • Im Django Admin: Benutzer auswählen → Löschen. Alle verknüpften Daten werden kaskadierend gelöscht.
  • Achtung: Prüfen Sie die gesetzliche Aufbewahrungspflicht (§16 ArbZG: 2 Jahre) vor der Löschung.
  • Löschung im Audit-Log dokumentieren (manueller Eintrag oder Django Admin Log).
Aufbewahrungspflicht (§16 ArbZG): Zeitaufzeichnungen müssen mindestens 2 Jahre aufbewahrt werden. Zeiteinträge und Audit-Log-Einträge dürfen vor Ablauf dieser Frist nicht gelöscht werden. Original-Zeiten bei Korrekturen bleiben dauerhaft in der Datenbank erhalten.
