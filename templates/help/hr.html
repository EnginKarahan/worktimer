Handbuch für Personalmanagement

Erweiterte Funktionen für Personalverantwortliche

1. HR-Dashboard

  • Das HR-Dashboard (Menü: „HR") zeigt eine Tagesübersicht: Wer ist heute abwesend, wer ist eingestempelt.
  • Offene Urlaubsanträge werden mit direkten Genehmigen/Ablehnen-Buttons angezeigt.
  • Mitarbeiter mit Minus- oder hohen Überstunden werden hervorgehoben.

2. Mitarbeiterverwaltung

  • Unter HR → Mitarbeiter können Sie alle aktiven Mitarbeiter einsehen, bearbeiten und anlegen.
  • Neue Mitarbeiter: Klicken Sie auf „+ Neuer Mitarbeiter", füllen Sie Basis- und Profildaten aus.
  • Mitarbeiter deaktivieren (DSGVO-konform): Setzen Sie den Status auf inaktiv. Daten bleiben erhalten.
  • Kontaktdaten (Telefon, Abteilung) werden im Profil gespeichert.

3. Zeiteinträge korrigieren

  • Im Mitarbeiterprofil unter „Zeiten" können Sie Zeiteinträge bis zu 3 Monate zurück korrigieren.
  • Jede Korrektur erfordert eine Begründung (Pflicht für §16 ArbZG Betriebsprüfung).
  • Original-Start-/Endzeiten werden gesichert und im Audit-Log protokolliert.

4. Urlaubsanträge genehmigen / ablehnen

  • Offene Anträge erscheinen im HR-Dashboard und unter HR → Offene Anträge.
  • Genehmigen: Mit Klick auf „Genehmigen" (optional: Kommentar).
  • Ablehnen: Klicken Sie auf „Ablehnen" und geben Sie einen Ablehnungsgrund an.
  • Der Mitarbeiter wird automatisch benachrichtigt.

5. Krankmeldung erfassen

  • Öffnen Sie das Mitarbeiterprofil und klicken Sie auf „Krankmeldung erfassen".
  • Geben Sie Start- und Enddatum ein. Die Krankmeldung wird sofort als genehmigt eingetragen.
  • Mitarbeiter können Krankmeldungen auch selbst über Abwesenheiten → Krank einreichen.

6. Berichte herunterladen

  • Im Mitarbeiterprofil können Sie monatliche Arbeitszeitnachweise als PDF oder Excel für beliebige Mitarbeiter herunterladen.
  • Wählen Sie Jahr und Monat, dann das gewünschte Format.
  • Downloads werden im Audit-Log protokolliert.
DSGVO-Hinweis (§26 BDSG): Alle personenbezogenen Daten werden ausschließlich zur Erfüllung arbeitsrechtlicher Pflichten (ArbZG, Entgeltfortzahlungsgesetz) verarbeitet. Mitarbeiterdaten dürfen nicht für andere Zwecke genutzt werden. Soft-Delete (Deaktivierung) wahrt das Löschrecht (Art. 17 DSGVO); endgültige Löschung nur durch Admin.
